任務運行間隔設置工具使用教程

一、功能簡介

本工具旨在避免自動化任務重複運行浪費時間，你可以設置任務的運行間隔。在未達到刷新時間時，系統會自動跳過該任務。工具支援每日、每週、每月三種模式，還能設置每個週期內的循環次數和最小間隔。

二、界面概述

打開任務管理界面後，找到「任務冷卻」相關設置區域，該區域包含以下主要部分：

  • 模式選擇：有每日、每週、每月三種模式的單選按鈕。
  • 時間設置：根據所選模式，提供不同的時間設置選項。
  • 循環設置：設置每個週期內任務最多運行次數和最小間隔。

三、詳細設置步驟

（一）每日模式設置

  1. 選擇模式：點擊「每日模式」單選按鈕。
  2. 設置刷新時間：在時間選擇區域，找到設置小時和分鐘的輸入框，輸入具體的刷新時間。例如，若要設置每天 8:00 刷新，就在輸入框内輸入 8。
  3. 設置循環次數和最小間隔：
    • 循環次數：在「循環次數」輸入框中輸入每個週期內任務最多運行的次數。例如，設置為 3 表示每天最多運行 3 次。
    • 最小間隔：在「最小間隔」輸入框中輸入任務每次運行的最小間隔時間（單位根據界面提示，可能是分鐘）。例如，設置為 60 表示任務每次運行至少間隔 60 分鐘。

（二）每週模式設置

  1. 選擇模式：點擊「每週模式」單選按鈕。
  2. 設置刷新日期：在星期選擇區域，選擇任務刷新的星期幾。例如，選擇「星期三」。
  3. 設置刷新時間：同每日模式設置刷新時間的方法，輸入具體的小時和分鐘。
  4. 設置循環次數和最小間隔：與每日模式設置方法相同。

（三）每月模式設置

  1. 選擇模式：點擊「每月模式」單選按鈕。
  2. 設置刷新日期：在日期輸入框中輸入任務刷新的具體日期，範圍為 1 - 31。例如，輸入 15 表示每月 15 號刷新。
  3. 設置刷新時間：輸入具體的小時和分鐘。
  4. 設置循環次數和最小間隔：與每日模式設置方法相同。

四、確認設置

完成上述所有設置後，點擊「確認」按鈕保存設置。系統會按照你設置的模式、刷新時間、循環次數和最小間隔來執行任務。

五、注意事項

  • 時間格式：請確保輸入的時間格式正確，小時範圍為 0 - 23，分鐘範圍為 0 - 59。
  • 日期範圍：每月模式中，輸入的日期要在 1 - 31 之間，系統會自動處理不同月份的日期差異。
  • 循環次數：循環次數設置為 0 時，任務在該週期內不會運行。
  • 最小間隔：最小間隔設置要合理，避免設置過短導致任務頻繁運行，設置過長導致任務無法按預期次數運行。
