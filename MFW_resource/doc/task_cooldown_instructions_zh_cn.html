任务运行间隔设置工具使用教程

一、功能简介

本工具旨在避免自动化任务重复运行浪费时间，你可以设置任务的运行间隔。在未达到刷新时间时，系统会自动跳过该任务。工具支持每日、每周、每月三种模式，还能设置每个周期内的循环次数和最小间隔。

二、界面概述

打开任务管理界面后，找到“任务冷却”相关设置区域，该区域包含以下主要部分：

  • 模式选择：有每日、每周、每月三种模式的单选按钮。
  • 时间设置：根据所选模式，提供不同的时间设置选项。
  • 循环设置：设置每个周期内任务最多运行次数和最小间隔。

三、详细设置步骤

（一）每日模式设置

  1. 选择模式：点击“每日模式”单选按钮。
  2. 设置刷新时间：在时间选择区域，找到设置小时和分钟的输入框，输入具体的刷新时间。例如，若要设置每天 8:00 刷新，就在输入框内输入 8。
  3. 设置循环次数和最小间隔：
    • 循环次数：在“循环次数”输入框中输入每个周期内任务最多运行的次数。例如，设置为 3 表示每天最多运行 3 次。
    • 最小间隔：在“最小间隔”输入框中输入任务每次运行的最小间隔时间（单位根据界面提示，可能是分钟）。例如，设置为 60 表示任务每次运行至少间隔 60 分钟。

（二）每周模式设置

  1. 选择模式：点击“每周模式”单选按钮。
  2. 设置刷新日期：在星期选择区域，选择任务刷新的星期几。例如，选择“星期三”。
  3. 设置刷新时间：同每日模式设置刷新时间的方法，输入具体的小时和分钟。
  4. 设置循环次数和最小间隔：与每日模式设置方法相同。

（三）每月模式设置

  1. 选择模式：点击“每月模式”单选按钮。
  2. 设置刷新日期：在日期输入框中输入任务刷新的具体日期，范围为 1 - 31。例如，输入 15 表示每月 15 号刷新。
  3. 设置刷新时间：输入具体的小时和分钟。
  4. 设置循环次数和最小间隔：与每日模式设置方法相同。

四、确认设置

完成上述所有设置后，点击“确认”按钮保存设置。系统会按照你设置的模式、刷新时间、循环次数和最小间隔来执行任务。

五、注意事项

  • 时间格式：请确保输入的时间格式正确，小时范围为 0 - 23，分钟范围为 0 - 59。
  • 日期范围：每月模式中，输入的日期要在 1 - 31 之间，系统会自动处理不同月份的日期差异。
  • 循环次数：循环次数设置为 0 时，任务在该周期内不会运行。
  • 最小间隔：最小间隔设置要合理，避免设置过短导致任务频繁运行，设置过长导致任务无法按预期次数运行。
